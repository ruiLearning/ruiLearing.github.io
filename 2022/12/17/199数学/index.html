199数学基础总结(基础点进去看👀)

199数学_计算力训练营_刘智👀

条件充分性判断-秒杀技巧

做题步骤

  1. 先分题型
    • 先确定联合型，再确定单独型
    • 如果联合型和单独型混淆，都视为单独型
    • 联合型全选C,单独型全选A
  2. 在做联合型
    1. 对称性
    2. 奇偶性
    3. 联合是否还缺条件，有缺改E
    4. 联合找反例，找到反例改E
  3. 最后做单独型
    1. 奇偶性
    2. 对称性
    3. 看看哪个条件与结论联系紧密，优先选联系紧密的
    4. 再分析有没有一正一负,等价条件，如果有改为D

Tips:如果条件是范围，取反例

三个大招

对称

  1. 交换条件中的变量，不改变条件，则称条件对称；
  2. 交换大前提中的变量，不改变大前提，则称大前提对称；
  3. 交换结论中的变量，不改变结论，则称结论对称；

解题步骤：

  1. 前提：结论要为能确定
  2. 条件对称,结论不对称 => 不能确定 => 排除
  3. 如何判断是否对称：交换条件/大前提/结论中的变量，结果不改变，则为对称

偶次方

没有限制正数，结论奇次方，条件偶次方->不充分。

反例

  1. 反例
    • 反(结论反向取值)+例(同时找满足条件的值)
    • 反例如果有值–>不充分
    • 反例如果没有值–>充分
  2. 范围
  3. 0的妙用 (要么取0，要么让它等于0)

判断题型

联合型

  1. C
    • 定性+定量
      • 定性：没有数字，列不出方程(不等式)
      • 定量：有数字，可以列出方程(不等式)
      • 等号(为、是、成 等)+不等号(大于、小于、多于、少于 等)：条件分别是等号和不等号
    • 结论有多个未知数，每个条件只有部分未知数
    • 结论有多个未知数，每个条件只有部分方程
  2. C/E型，优先选C
    • 先选C,之后检查4次(二次验证)
      1. 奇偶性
      2. 对称性
      3. 联合起来有反例
      4. 联合起来还缺不缺
  3. E
    • 联合起来还缺不缺
    • 联合起来有反例

单独型

  1. A/B型
    • 一字之差、两字之差 且无法联合
    • 一念之差(概念)
    • 一围之差(范围)
    • 一系之差(不等关系)
  2. A/B型优先选A,在检查4次
    1. 奇偶性
    2. 对称性
    3. 举条件2的反例，没有反例则选B,有反例则选A。如果还有时间，在举条件1的反例。
  3. D型
    • 两条件一正一负
    • 两条件等价

条件充分性判断-专项突破

1.概念

  1. 判断所给出的条件 能否 充分支持 题干中陈述的结论。
  2. 条充 => 恒成立 （判断恒成立）
  3. ️⭐️⭐️⭐️只可从条件推结论。⭐️⭐️⭐
  4. 结论只可以等价变换。
  5. ️⭐️⭐️⭐️条件为结论的子集必充分，即子集充分。⭐️⭐️⭐️
  6. 例子说明
    如果能从条件中找到一个反例，则推不出结论恒成立，故条件不充分。反例：一票否决。
    1. 特例: 满足条件，称为特例。
    2. 正例: 满足条件，也满足结论,称为正例。
    3. 反例: 满足条件，不满足结论,称为反例。

2.破题方向

等号 > 复杂的 > 简单的 > 不等号

  1. 没有大前提，如果条件比结论复杂，先从复杂的条件➡️➡️➡️(推出)结论。
  2. 没有大前提，如果结论比条件复杂，先从复杂的结论➡️ 等价变换。
  3. 条件是等号，且方便带入计算的，先从条件➡️➡️➡️(推出)结论。
  4. 有大前提，如果条件比结论复杂，大前提 + 条件 ➡️➡️➡️(推出)结论。
  5. 有大前提，如果结论比条件复杂，大前提 + 结论(等价变换)，化简后再用条件➡️➡️➡️(推出)结论。
  6. 条件(1)和条件(2)有公共条件，可以提取出来当做大前提。

3.建立联系

等差数列

三步走

  1. 中值=均值 公式2，,9，,10
    (两个条件)化简多个a为一个a$\begin{cases}
    b = \frac{a+c}{2}\\
    a_{\frac{n+m}{2}} = \frac{a_n+a_m}{2}\\
    a_{\frac{1+n}{2}} = \frac{S_n}{n}
    \end{cases} $
    (一个条件)=> 特值
  2. 有两个a,求$d=\frac{a_n-a_m}{n-m}$ 公式7
  3. $a_n = a_m + (n-m)d $ 公式4 盖烂尾楼

基本公式

  • 公式1 $a_n - a_{n-1} = d(n\geq 2) $等差数列的判定法则1
  • 公式2 $ a,b,c成等差数列 ↔️ b = \frac{a+c}{2}$
  • 公式3 $a_n = a_1+(n-1)d $
    • 公式5 $a_n = d·n + (a_1-d) = k·n + b $等差数列的判定法则2
    • 公式6 $n = \frac{a_n-a_1}{d} + 1 $ 计算项数
  • 公式4 $a_n = a_m + (n-m)d $ 盖烂尾楼
    • 公式8 $a_n+a_m = a_x+a_y
      \begin{cases}
      项数相同\\
      n+m = x+y
      \end{cases} $ 中值即均值
    • 公式9 $a_{\frac{n+m}{2}} = \frac{a_n+a_m}{2} $
    • 公式10 $S_n = \frac{(a_1+a_n)n}{2} = na_{\frac{1+n}{2}} ↔️ a_{\frac{1+n}{2}} = \frac{S_n}{n}$中值等于均值
    • 公式11 $S_n = \frac{d}{2}·n^2 + (a_1-\frac{d}{2})n + 0 = An^2+Bn+0 $ 等差数列的判定法则3
      • $当n = \frac{1}{2}-\frac{a_1}{d}时，S_n有最值$
  • 公式7 $d = \frac{a_n-a_m}{n-m} = \frac{高度差}{层差} $ 2个a求公差 计算公差

分类

  1. 递推公式(定性)
    • 公式1 $a_n - a_{n-1} = d(n\geq 2) $等差数列的判定法则1
    • 公式2 $ a,b,c成等差数列 ↔️ b = \frac{a+c}{2}$
    • 公式5 $a_n = d·n + (a_1-d) = k·n + b $等差数列的判定法则2
    • 公式11 $S_n = \frac{d}{2}·n^2 + (a_1-\frac{d}{2})n + 0 = An^2+Bn+0$ 等差数列的判定法则3
      • $当n = \frac{1}{2}-\frac{a_1}{d}时，S_n有最值$
  2. 计算公式(定量)
    • 公式7 $d = \frac{a_n-a_m}{n-m} = \frac{高度差}{层差} $ 计算公差
    • 公式4 $a_n = a_m + (n-m)d ,a_m{d \choose ->}a_n $ 盖烂尾楼
    • 公式9 $a_{\frac{n+m}{2}} = \frac{a_n+a_m}{2} $ 化简
    • 公式10 $S_n = \frac{(a_1+a_n)n}{2} = na_{\frac{1+n}{2}} ↔️ a_{\frac{1+n}{2}} = \frac{S_n}{n}$中值等于均值
  3. 其它
    • 公式3 $a_n = a_1+(n-1)d $
    • 公式6 $n = \frac{a_n-a_1}{d} + 1 $ 计算项数
    • 公式8 $a_n+a_m = a_x+a_y $ 中值即均值

等比数列

基本公式

  • 公式12 $\frac{a_n}{a_{n-1}} = q(n\geq 2)\Rightarrow
    \begin{cases}
    a_n\not=0\\
    q\not=0
    \end{cases}$ 等比数列的判定法则1
  • 公式13 $ a,b,c成等比数列 ↔️
    \begin{cases}
    a\not=0,b\not=0,c\not=0\\
    b^2 = ac
    \end{cases}$
  • 公式14 $a_n = a_1·q^{n-1}$ 等比数列通项公式 列出a1,q,n这三要素
  • 公式15 $a_n = a_m·q^{n-m}$ 等比数列通项公式 列出a1,q,n这三要素
  • 公式16 $a_n = \frac{a_1}{q}·q^n = A·q^n $ 等比数列的判定法则2
  • 公式17 $a_n·a_m = a_x·a_y
    \begin{cases}
    项数相同\\
    n+m = x+y
    \end{cases} $
  • 公式18 $等比数列{a_n}中，”同奇偶，同正负” $
  • 公式19 $ S_n =
    \begin{cases}
    a_1\frac{1-q^n}{1-q}, q\not=1 \\
    na_1,q=1\\
    \approx \frac{a_1}{1-q} ,|q|<1
    \end{cases} $等比数列求和公式 列出a1,q,n这三要素
  • 公式20 $S_n = -\frac{a_1}{1-q}q^n + \frac{a_1}{1-q} = Cq^n - C $ 等比数列的判定法则3

分类

  1. 递推公式(定性)
    • 公式12 $\frac{a_n}{a_{n-1}} = q(n\geq 2)\Rightarrow
      \begin{cases}
      a_n\not=0\\
      q\not=0
      \end{cases}$ 等比数列的判定法则1
    • 公式16 $a_n = \frac{a_1}{q}·q^n = A·q^n $ 等比数列的判定法则2
    • 公式20 $S_n = -\frac{a_1}{1-q}q^n + \frac{a_1}{1-q} = Cq^n - C $ 等比数列的判定法则3
  2. 计算公式(定量)
    a三要素<1,q,n>
    • 公式14 $a_n = a_1·q^{n-1}$ 等比数列通项公式 列出a1,q,n这三要素
    • 公式15 $a_n = a_m·q^{n-m}$ 等比数列通项公式 列出a1,q,n这三要素
    • 公式19 $ S_n =
      \begin{cases}
      a_1\frac{1-q^n}{1-q}, q\not=1 \\
      na_1,q=1\\
      \approx \frac{a_1}{1-q} ,|q|<1
      \end{cases} $等比数列求和公式 列出a1,q,n这三要素
  3. 其它

    既是等差数列又是等比数列

  • 公式21 既是等差数列又是等比数列即非零常数列

既非等差数列又非等比数列

  • 公式22 $a_n =
    \begin{cases}
    S_n - S_{n-1}, n\geq 2 \\
    S_1,n=1\\
    \end{cases}$

    • $已知S_n求a_n,适用于所有数列$

  • 公式23 $\frac{1}{n·(n+1)} = \frac{1}{n} - \frac{1}{n+1}$ 列项公式

4.知识就是力量

必备公式

两个点，$A(x_1,y_1),B(x_2,y_2)$

  1. 两点间距离公式：$|AB| = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2}$

  2. 中点距离公式：$m(\frac{x_1+x_2}{2},\frac{y_1+y_2}{2})$

  3. 斜率公式：$k=\frac{y_2-y_1}{x_2-x_1}$

  4. 圆的一般方程式：
    $\begin{split}
    & (x-a)^2+(y+b)^2 = r^2 =>(a,b),r\\
    &\\
    &x^2+y^2+Dx+Ey+F=0 若F=0,即x^2+y^2+Dx+Ey=0,此圆必过坐标原点(0,0)\\
    &圆心坐标 = 负一半(-\frac{D}{2},-\frac{E}{2}),半径 r =\sqrt{ (-\frac{D}{2})^2+(-\frac{E}{2})^2-F} 圆心坐标的勾股数\\
    \end{split}
    $

  5. 点到直线距离公式：
    $\begin{split}
    & 点A(x_0,y_0)\\
    & 直线方程式：Ax+By+C=0\\
    &\\
    &点到直线距离d = \frac{点A带入直线方程式|Ax_0+By_0+C|}{\sqrt{A^2(x的方向系数)+B^2(y的方向系数)}}\\
    & = \frac{带入的绝对值}{方向系数的勾股数}
    \end{split}
    $

  6. $若半径 = 1， 2π·1=360^\circ => π=180^\circ$
    $\begin{split}
    & a^2+b^2 = c^2直角 \\
    & a^2+b^2 > c^2锐角 \\
    & a^2+b^2 < c^2钝角 \\
    & 斜率规律\\
    & 斜率k = \frac{y_2-y_1}{x_2-x_1} = \frac{y_差}{x_差} \\
    & 斜率k > 0,x往右，y轴上升 \\
    & 斜率k = 0,平行于x轴 \\
    & 斜率k > 0,x往右,y轴下降 \\
    & 常用角度斜率，及互补斜率\\
    & 30^\circ,k = \frac{1}{\sqrt{3}}, 150^\circ,k = -\frac{1}{\sqrt{3}} \\
    & 45^\circ,k = 1, 135^\circ,k = -1 \\
    & 60^\circ,k = \sqrt{3}, 120^\circ,k = -\sqrt{3} \\
    & \\
    & 结论两个：\\
    & ①两角互补，斜率相反 \\
    & ②两直线垂直，K_1·K_2 = -1 \\
    \end{split}
    $

  7. 象限

    • 必过第1象限：k>0 或 b>0 ,必不过第1象限：k<=0 且 b<=0
    • 必过第2象限：k<0 或 b>0 ,必不过第2象限：k>=0 且 b<=0
    • 必过第3象限：k<0 或 b<0 ,必不过第3象限：k<=0 且 b>=0
    • 必过第4象限：k<0 或 b<0 ,必不过第4象限：k>=0 且 b>=0

  8. 5类特殊对称

    • (a,b)关于原点(0,0)对称，(-a,-b)
    • (a,b)关于x对称，(a,-b)
    • (a,b)关于y对称，(-a,b)
    • (a,b)关于y=x对称，(b,a)
    • (a,b)关于y=-x对称，(-b,-a)

  9. 横过定点 ⭐️⭐️⭐️⭐️⭐️
    解关于x的方程ax=b
    $
    \begin{cases}
    x=\frac{b}{a},a\not=0 \\
    x\in R,a=0,b=0 \\
    x\in \varnothing, a=0,b \geq 0
    \end{cases}
    $

    $\\
    \begin{aligned}
    &\because 0乘以任何数等于0 \\
    &\therefore 当a=0且b=0时，方程ax=b的解为全体实数。\\
    &即不论x取何值，ax=b恒成立 ↔️ a=0且b=0
    \end{aligned}$

  • 斜率含参，恒过定点
    • 例子
      $y = kx+2k => (x+2)k = y$
      当$
      \begin{cases}
      x+2=0 \\
      y=0
      \end{cases}
      $ 时，方程对一切k值恒成立
      不论k取何值，直线恒过(-2,0)定点
