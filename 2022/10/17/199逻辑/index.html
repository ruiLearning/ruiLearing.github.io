秒杀

如果、则的转换(基本秒杀公式)

  1. 如果p=才p=只要p=所有p=则非P=只有(非P)=或者(非P)=除非(非P)
  2. 则p=只有p=必须p=除非p=如果非P=才非P=或者p=那么p=就p
  3. 如果p，则q = 如果非q，则非p
  4. 只有P，才q = 如果q 则p
  5. 或者p，或者q =如果非p，则q = 如果非q，则p
  6. 除非p，否则q = 如果非p，则q = 如果非q，则p = 或者p，或者q=只有P，才q
    如果p则q为真

排除法

如果p=才p=只要p=所有p=则非P=只有(非P)=或者(非P)=除非(非P)
则p=只有p=必须p=除非p=如果非P=才非P=或者p=那么p=就p

如果=只要=才=所有
则=除必=只有=就=那么=或着

排除不符合推理规则的

  • 根据如果p，可以排除:(题干条件中p仅出现1次)

    • 则p
    • 只有p
    • 除非p
    • 或者p
    • 如果非p
    • 如果p则q为真

  • 根据则q，可以排除:(题干条件中q仅出现1次)

    • 如果q
    • 则非q
    • 只有非q
    • 或者非q
    • 除非非q

推理

易混推理总结

  1. 不是p就是q=不是q就是p=p或q
  2. 不是p而是q=非p且q
  3. 除非P，否则q=非P则q
  4. 若要A，除非B=如果A则B
  5. A，除非B=除非B否则A
  6. 只要p就q=如果p则q
  7. p和q至少选一个=p或q
  8. p和q至多选一个=p和q至少有一个不选=非p或非q
  9. 不选甲和乙=不选甲 且 不选乙
  10. 不选甲且乙=不甲 或 不乙
  11. 不选甲或乙=不选甲 且 不选乙
  12. 甲和乙都不选=不甲 且 不乙
  13. 甲和乙不都选=不甲 或 不乙
  14. 不同时选甲和乙=不甲 或 不乙
  15. 不同时选甲或乙=不甲 且 不乙

推理关键词

所有:都，凡，任何，没有一个是，没有一个不是，每个，每一个
有的:有些，部分，少数，多数，大多数
且:而，而且，和，但，既...又...，不但，而且
或:或者，或者...或者，至少一个，至多一个
要么:要么p，要么q，二者择其一，有且只有一个
如果，只要，就，则，那么
只有，才，除非，否则，必须
可能，必然，一定

推理规则

  1. p或q的推理规则

    1. 非P ➡ q(如果非 p 则 q)
    2. 非q ➡ p(如果非 q 则 p)
    3. 不是p就是q = p或q = 非p ➡ q = 非q ➡ p
    4. p或q = 非p ➡ q(否定一边，肯定另一边)
    5. p或q = 非q ➡ p(否定一边，肯定另一边)

  2. 如果p则q的推理规则

    1. 如果P则q
      =P→q(肯前则肯后)
      =非q→非p(否后则否前)

  3. 直言

    1. 有的s是p≠有的s不是p
    2. 有的s是p=有的p是s；有的s不是p=有的非p是s
    3. 所有s不是p=所有p不是s
    4. 所有s是p=有的p是s

  4. 矛盾

    1. 所有 与 必然
    2. 可能 与 必然
    3. 且 与 或
    4. 如果p则q 与 p且非q
    5. 要么p,要么q 与(p且q) 或(非p且非q)

  5. 直言判断的负判断/矛盾判断

    • 并非所有S都是P ——-有的S不是P
    • 并非所有S都不是P ——-有的S是P
    • 并非有的S是P ——-所有S都不是P
    • 并非有的(不是S)的是P——-所有(不是S)的不是P=所有P是S(否后否前)
    • 并非有的S不是P ——-所有S都是P
    • 并非这个S是P ——-这个S不是P
    • 并非这个S不是P ——-这个S是P

  6. 或的转换

    1. p，q至少一个成立=P或q
    2. P，q至少一个不成立=非p或非q
    3. P，q至多一个成立=p，q至少一个不成立=非p或非q
    4. P，q至多一个不成立=p，q至少一个成立=p或q
    5. p，q，r至少一个成立=P或q或r
    6. p，q，r至少一个不成立=非P或非q或非r

  7. 如果的转换

    1. 所有p是q=如果p则q；
    2. 所有p不是q=如果p则非q
    3. 只有p才q=如果q则p；
    4. 除非p才q=如果q则p
    5. 要p成立，(就)必须q成立=如果p成立，则q成立
    6. 必须p成立，才q成立=只有p成立才q成立=如果q成立则p成立
    7. 只要p就q=如果p则q；
    8. 若要p，除非q=如果p，则q
    9. 除非p，否则q=如果非p则q；
    10. 除非P，才q=只有P才q；
    11. q，除非p=除非p，否则q=如果非p则q=P或q；
    12. p或q=如果非p则q=如果非q则p；
    13. 如果p则q=非p或q
    14. 除非p=或者p
    15. 除非p，否则q=或者p，或者q

  8. 所有的转换

    1. 所有s是p(s→p)=所有不是p的不是s(不是p→不是s)
    2. 所有s不是p(s→不是p)=所有p不是s(p→不是s)
    3. 没有一个s是p=所有s不是p
    4. 没有一个s不是p=所有s是p
    5. 其他表示所有的词:任何、所有、都、凡、每一个

  9. 有的转换

    1. 有的s是p=有的p是s；
    2. 有的s是p⇏有的s不是p
    3. 有的s不是p⇏有的p不是s
    4. 有的s不是p=有的s是非p=有的非p是s
      对于上述特称命题，s，p可以换位置，但符号不能改变。

  10. 复合命题的矛盾命题

    • 并非p且q=非p或非q
    • 并非p或q =非p且非q
    • 并非(要么p，要么q)=(p且q)或(非p且非q)
    • 并非(如果p则q)= p且非q

    只有P才q的负判断为并非(只有P才q)
    并非(只有P才q)=非(q→P)=q且非p
    p当且仅当q的负判断为(p且非q)或(q且非p)
    非(p当且仅当q)=(p且非q)或(q且非p) =要么p要么q
    如果p则q的矛盾为非(p→q)=p且非q

    模态判断的负判断
    不可能P=必然非P 不可能非P=必然P
    不必然P=可能非P 不必然非P=可能P
    若要P除非Q=如果P则Q
    若要有一流实力，除非有一流教育=如果有一流实力→则有一流教育
    除非P否则Q=非P则Q
    (则)P，除非Q=除非Q，则P
    冬天会很冷，除非有暖气

    =除非有暖气→冬天冷
    =有暖气或者冬天冷
    =没有暖气→冬天冷
    =冬天不冷→有暖气

  11. 考题中十组常见的矛盾公式

    1. 所有S是P —–有些S不是P
    2. 所有S不是P ——有些S是P
    3. 这个S是P ——这个S不是P
    4. 必然P ——可能非P
    5. 可能P ——必然非P
    6. P且Q ——非P或非Q
    7. P或Q ——非P且非Q
    8. 要么P，要么Q ——(P且Q)或(非P且非Q)
    9. 如果P，那么Q ——P且非Q
    10. 只有P才Q ——非P且Q

  12. 考题中六组常见的反对公式

    1. 所有S是P ——所有S不是P
    2. 所有S是P ——这个S不是P
    3. 所有S不是P ——这个S是P
    4. P且Q ——非P
      P且Q ——非Q
      P且Q ——非P且非Q
    5. 如果P则Q —–如果P则非Q
    6. 只有P才Q(Q —＞P) —–没有P也有Q(Q且非P)

    如果的矛盾是且
    所有的矛盾是有的
    有的s是p⇏有的s不是p
    有的s不是p⇏有的p不是s
    有的s是p=有的p是s

论证

削弱、加强、假设(未知、隐含)、评价评论方法、评价谬误、评价争论焦点

  1. 会思考答案
  2. 会排出迷惑选项

论证其它重点

从结论看论证方法

结论同时出现因和果，论证方法几乎都是 因 ➡️️ 果(找原因 找原因 找原因)
虽然A和B有一定的关系，但是A不必然导致B，影响B的因素除了A之外还有C、D、E、F，所以我们不能简单的认为B的原因就是A。

因果

  1. 因果分不清
  • 已发生(以前/现在/之前) = 因
  • 未发生(以后/未来/将来) = 果
  1. 出现因和无果
  • 结论同时出现因和果的没有，那么只有一种考法 ➡️ 无他因️

论证的假设

有的题目会有隐含假设

  1. 论证成立的必要条件
  2. 从论证可以/最可能推出
  3. 该假设不成立 削弱论证

论证必记考点

  • 归纳
    • 数量
    • 代表性
    • 反例
  • 类比
    • 可比性(相关属性是否相同)
  • 因到果
    • 关系
    • 他因
  • 果到因
    • 关系
    • 他因
    • 倒置
    • 无因有果
  • 对比推因果
    • 可比性(相关属性是否相同)
    • 因果是否成立
  • 措施
    • 可行性
    • 能否达到目的
    • 是否产生严重后果

论证练习 x10

例题:
北清中学参加象棋比赛的比不参加象棋比赛的同学数学平均分高，因此，北清中学的学生参加象棋比赛有利于提高数学成绩。

削弱
(1)参加象棋比赛的同学比不参加象棋比赛的同学基础好。(不可比)
(2)参加象棋比赛的同学和不参加象棋比赛的同学基础不同。(不可比)
(3)数学成绩好的同学更可能去参加象棋比赛。(不可比)
(4)因为数学成绩好才去参加象棋比赛。(因果倒置)
(5)只有数学成绩好的才能参加象棋比赛/参加象棋比赛要是数学好为前提条件。(因果倒置)
(6)象棋比赛和数学成绩之间的关系不大。(因果无关)
(7)很多北清中学的学生参见了象棋比赛，但是数学成绩没有提高。(因果无关)
(8)北清中学参加象棋比赛的同学大多参见了校外的数学辅导班。(他因)
(9)北清中学参加象棋比赛的同学如果不参加，也能提高数学成绩。(无因有果)
(10)和北清中学各方面情况类似的南青中学的学生没有参加象棋比赛也提高了数学成绩。(无因有果)
加强
(1)参加象棋比赛的同学和不参加象棋比赛的同学基础相同/差不多/非常类似。(可比)
(2)参见象棋比赛的同学并不比不参加象棋比赛的同学基础好。(可比)
(3)数学成绩不好的同学更可能去参见象棋比赛/参加象棋比赛的同学比不参加象棋比赛的同学基础差。(可比)
(4)不是因为数学成绩好才去参加象棋比赛。(参见象棋比赛并不以成绩好为前提)(没有倒置)
(5)象棋比赛有助于提高智力，智力高则数学成绩就容易提高。(因果相关)
(6)很多北清中学的学生参加了象棋比赛，数学成绩明显提高。(因果相关)
(7)北清中学参见象棋比赛的同学并非因为基础好(或数学成绩好并非因为参加了校外数学辅导)(不是他因)
(8)北清中学的参加象棋比赛的同学如果不参见象棋比赛就不会提高数学成绩。(无因无果)
(9)和北清中学各方面情况类似的南青中学没参见象棋比赛的学生数学成绩没有提高。(无因无果)
假设
(1)参加象棋比赛的同学并不比不参加象棋比赛的同学基础好。(可比)
(2)参加象棋比赛的同学和不参加象棋比赛的同学基础相同/差不多/非常类似。(可比)
(3)不是因为数学成绩好才去参加象棋比赛。(参加象棋比赛并不以成绩好为前提)(没有倒置)
(4)参加象棋比赛有助于提高数学成绩。(因果相关)
(5)至少有些北清中学的学生参加象棋比赛，数学成绩提高了。(因果相关)
(6)北清中学参加象棋比赛的同学成绩好并非因为基础好(或数学成绩好并非因为参加了校外的数学辅导)(不是他因)
(7)北清中学的参加象棋比赛的学生如果不参加象棋比赛就不会比不参加象棋比赛的同学数学考试成绩高(或不能提高数学成绩)(无因无果)

措施论证

措施论证-削弱题型
常见迷惑选项

  1. 措施不合理
  2. 措施有副作用
  3. 措施花费高，价格贵
  4. 其他措施效果好或一样好
  5. 措施只能达到这个目的，不能达到其他目的
  6. 措施不能防止某个严重后果
  7. 把措施或目的偷换概念

答案:

  1. 不可行
  2. 达不到目的
  3. 严重后果
  4. 问题不严重(问题 ➡️️ 措施)

论证识别

归纳论证: 部分 ➡️️ 全部
类比论证:个别 ➡️️ 个别; 相同 ➡️️ 相同
因果论证:

  1. 因 ➡️️ 果
  2. 果 ➡️️ 因

对比论证:对比 ➡️️ 因果
措施论证:

  1. 目的 ➡️️ 措施
  2. 问题 ➡️️ 措施
